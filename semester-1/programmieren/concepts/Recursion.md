Wie viele Z체ge sind notwendig?

$A_{1} = 1$
$A_{n+1}= 2\;A_{n} + 1$

### Implementation
[[hanoi.c]]

Pseudo-Code:
```python
def hanoi(from, to, disc):
	temp_pos = position \ {from, to}
	if disc == 1:
		move(disc, from, to);
	
	hanoi(from, temp_pos, disc - 1)
	move(disc,from, to)
	hanoi(temp_pos, to, disc - 1)
```

to get temp_pos index:
```python
temp_pos = (1 + 2 + 3) - (from + to)
```
